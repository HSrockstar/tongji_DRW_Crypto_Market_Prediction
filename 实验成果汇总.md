## 二、本地指标说明

- **Holdout**：train 末 20%，与 test 分布不一致，**不可用于选模/加权**。
- **Purged CV**：6 折 + gap=1，较 Holdout 可靠，但仍偏高（~0.10）。
- **pre-window Pearson**：3mo 训练窗之前的参考段，~0.021，**最接近 Kaggle Private**。

## 三、EDA 主要结论（`outputs/eda/`）

- 信号主要在匿名 X（单特征 max |corr| ≈ 0.09）。
- X 高度共线（最大簇 ~565 个），medoid 硬删风险高。
- train/test 成交量均值差 ~60%，market 特征 OOD 不稳。

## 五、失败经验（写报告可用）

1. **Holdout 0.10 ≠ Kaggle 0.02**：test 为 2024.02 后数据，存在 regime shift。
2. **勿用 Holdout 搜 blend 权重**：Step3 本地 0.113 → Kaggle 0.010。
3. **medoid 删特征**：Purged CV 仅 ~0.02，Holdout 虚高后崩盘。
4. **高分方向（未实现）**：Top-X 交互特征 + MLP(Pearson loss) + 多窗集成。
